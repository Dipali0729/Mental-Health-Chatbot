# Mental Health AI Chatbot

## Overview

The Mental Health AI Chatbot is an intelligent conversational application designed to provide emotional support and basic mental wellness guidance. It uses AI to interact with users in a natural and empathetic manner, helping them express their thoughts and feel heard.

This project integrates backend logic, a user-friendly interface, and a locally running AI model to deliver real-time responses without relying on external APIs.

---

## Technologies Used

* Python – Core programming language
* Flask – Backend framework to handle API requests and chatbot logic
* Streamlit – Frontend interface for interactive UI
* Ollama – Runs the local AI model for generating responses

---

## Features

* AI-powered conversations with context-aware responses
* Clean and simple user interface
* Real-time interaction
* Privacy-focused using a local AI model
* English language support

## How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Dipali0729/Mental-Health-Chatbot.git
cd Mental-Health-Chatbot
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the backend

```bash
python app.py
```

### 4. Run the frontend

```bash
streamlit run ui.py
```
